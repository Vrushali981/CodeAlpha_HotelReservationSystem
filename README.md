# Hotel Reservation System

A Java program to book and manage hotel rooms with payment simulation.

## Features
- View available rooms (Standard, Deluxe, Suite)
- Make and cancel reservations
- Payment simulation (Card, Cash, UPI)
- Save and load bookings from file

## How to Run
```
javac HotelReservationSystem.java
java HotelReservationSystem
```

## Tech Used
Java, OOP, File I/O, LocalDate, ArrayList
